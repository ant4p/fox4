Установка и устранение проблем Fail2ban на Ubuntu 22.04 LTS

Fail2ban

Установка fail2ban:

sudo apt update
sudo apt upgrade

sudo apt install fail2ban

С версии 20.04 LTS в Ubuntu необходимо немного изменить конфигурационный файл, чтобы fail2ban запустился:

sudo vim /etc/fail2ban/jail.d/defaults-debian.conf
В записи [sshd] добавляем строку:
backend = systemd

sudo systemctl start fail2ban

Для проверки:
sudo systemctl status fail2ban

Первоначально в fail2ban уже стоят необходимые настройки безопасности. Отдельно можно не конфигурировать.
