Использование зеркал для Docker

Если докер снова перестал работать в РФ.


Настраиваем закачку образов с зеркала Google

Подключаемся по ssh к серверу, затем:

sudo vim /etc/docker/daemon.json

Если файл пустой, то добавляем ключ registry-mirrors с зеркалом Google в конфигурацию.

{
    "registry-mirrors": [
    "https://mirror.gcr.io"
  ]
}

После чего - перезапускаем Docker:
sudo systemctl restart docker

Как вариант можем добавить ещё зеркала сторонних компаний, но зеркала Google должно хватить.
